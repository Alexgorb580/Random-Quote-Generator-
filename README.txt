Random Quote Generator (Генератор случайных цитат)
Автор: Горбачев Александр Владимирович
Описание: Простое графическое приложение на Python (Tkinter), которое:

хранит предопределённый список цитат (текст, автор, тема);
генерирует случайную цитату по кнопке;
сохраняет историю сгенерированных цитат в файл JSON и загружает её при старте;
позволяет фильтровать историю по автору и по теме.
Что нужно:

Python 3.x
Tkinter (идёт в комплекте с Python)
Файлы проекта:

main.py — основная программа и графический интерфейс
quotes_data.py — предопределённый список цитат (text, author, theme)
quotes_history.json — история сгенерированных цитат (создаётся автоматически)
Как запустить:

Скачайте или распакуйте проект в папку на компьютере.
Откройте терминал (или командную строку) и перейдите в папку проекта.
Запустите: python main.py
Как пользоваться:

В окне нажмите кнопку «Сгенерировать цитату» чтобы получить случайную цитату.
В верхней части можно выбрать фильтры:
Автор: выбрать конкретного автора или «Все»
Тема: выбрать тему или «Все» Фильтры применяются к отображению истории цитат.
История цитат отображается как список внизу окна. история сохраняется в файл quotes_history.json и подгружается при каждом запуске программы.
Редактирование цитат (как добавить новые):

Откройте quotes_data.py и добавьте новые цитаты в список quotes в формате: {"text": "Текст цитаты", "author": "Имя Автора", "theme": "Тема"}
После сохранения файла и перезапуска программы новые цитаты станут доступны для генерации.
Безопасность:

Генерация случайной цитаты использует модуль random и не является криптографически безопасной. Это нормально для учебного проекта.

Примеры использования:

Генерация без фильтров:
Нажать «Сгенерировать цитату» и увидеть случайную цитату из всех.
Фильтрация по автору и теме:
Выбрать конкретного автора и/или тему и сгенерировать цитату в рамках выбранных фильтров.
Просмотр истории:
История собирается автоматически и сохраняется в quotes_history.json. При повторном запуске истории будут доступны снова.
Примечания:

История сохраняется локально на вашем устройстве в файле quotes_history.json.
Если файл не создаётся автоматически, программа может не иметь прав на запись в папку — проверьте права доступа.
